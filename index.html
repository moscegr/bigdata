Panorama General

¿Qué es realmente Big Data?

No se trata solo de tener "muchos datos". Big Data es el conjunto de tecnologías y estrategias que permiten capturar, gestionar y procesar información que supera la capacidad de las herramientas de software habituales.

Más allá de SQL

Aprendemos a usar bases de datos NoSQL (como MongoDB) y frameworks como Hadoop para manejar datos no estructurados (videos, logs, redes sociales).

Toma de Decisiones

Transformamos datos crudos en Inteligencia de Negocios (BI). El objetivo final no es almacenar, sino predecir y optimizar.

El Reto de las 5 V's

Haz clic en las tarjetas para descubrir qué característica ("V") de Big Data describe cada situación del mundo real.

"Facebook procesa más de 500 Terabytes de datos nuevos cada día."

(Haz clic para ver)

VOLUMEN

Se refiere a la cantidad masiva de datos generados. Las herramientas tradicionales no pueden almacenar tal magnitud.

"La Bolsa de Valores analiza miles de transacciones en milisegundos."

(Haz clic para ver)

VELOCIDAD

La rapidez con la que los datos son generados, procesados y analizados. ¡A veces en tiempo real!

"Analizar textos de Twitter, imágenes de Instagram y audios de WhatsApp a la vez."

(Haz clic para ver)

VARIEDAD

Datos estructurados (tablas), semi-estructurados (JSON) y no estructurados (video/audio).

"Filtrar noticias falsas (Fake News) o datos erróneos de sensores GPS dañados."

(Haz clic para ver)

VERACIDAD

La fiabilidad y calidad de los datos. Datos "sucios" llevan a decisiones equivocadas.

"Usar los datos para aumentar las ventas un 20% este trimestre."

(Haz clic para ver)

VALOR

El fin último de Big Data. Los datos son inútiles si no se convierten en beneficio o conocimiento.

Tip: Estas "5 V's" son la base teórica de la Unidad 1.
